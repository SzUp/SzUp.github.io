单词系列（1）——前缀篇

单词 词缀
英语
发布日期:   2021-02-15
更新日期:   2021-02-15
文章字数:   2k
阅读时长:   8 分
阅读次数:  

这是我的第一篇博客，用来练习markdown的语法。以及把近一年的英语学习经验和成果做成一个系列分享给大家，希望对大家有帮助，谢谢。

当然，这仅仅是我自己的理解，有什么写的不妥或者不对的地方，欢迎在评论区指出，Thanks♪(･ω･)ﾉ！

1. pro

pro表示在前或者向前的意思，更多表示空间上的前。

例词：

program = pro 向前 + gram 写 = 向前写 → 程序，计划

provide

progress

2. pre

pre也是表示在前或者向前的意思，更多表示时间上的前。

例词：

prefer = pre 在前 + fer 拿 = 提前拿 → 许多东西选择，提前拿表示喜欢 → 喜欢

previous

present

3. post

post表示在后或者向后的意思。

例词：

postgraduate = post 后 + graduate 毕业 = 毕业后 → 研究生

posterior

postpone

4. per

per表示完全的意思。

例词：

perfect = per 完全 + fect 做 = 完全做好（强迫症） → 完美

perspective

perform

5. co族

co族中，每个成员都表示强调语气或者是在一起的意思。co族是个大家族，有下面这些成员：

  • co：

    例词：

    covariance = co 在一起 + var 变化 + i + ance = 在一起变化 → 协方差

    coincidence

  • com：

    例词：

    computer = com 在一起 + put 计算 + er 表示人或者物 → 在一起计算的东西（可以理解为处理器） → 计算机

    complete

    community

  • con：

    例词：

    confidence = con 在一起 + fid 相信 + ence = 互相信任 → 信任，引申为信心

    content

    conduct

  • co(*)：

    这个前缀比较特殊，有一个不确定成员 (*) 在，它决定于后面所跟随的第一个字母，下面举几个例子，相信能够帮助你更快理解。

    例词：

    collapse = co l 强调 + l aps 塌 + e = 倒塌，昏倒

    correlation = co r 强调 r elate 关系 = 相关

6. i(@)族

i(@)族中，共有in、im、i()三个成员。可能表示*向内、否定、使动中的其中几个意思。

  • in：

    例词：

    influence = in 向内 + flu 流 + ence = 向内流，引申为说到心坎上，影响到你 → 影响

    individual = in 否定 + divid 分割 + u + al = 不可分 → 个体

    indicate = in 使动/向内 + dic 说 + ate = 指示

  • im：

    例词：

    import = im 向内 + port 拿 = 进口

    impossible = im 否定 + possible 可能 = 不可能

    implement = im 向内/使动 + ple 完全 → 使…完全 → 做好准备 → 实施，执行

  • i(*)：

    此处的规律和co( )相同，决定于后面词根的首字母。

    例词：

    irregular = ir 否定 + regular 常规 = 非常规

    illegal = il 否定 + legal 合法 = 非法

    illustrate = il 强调 + lustr 光 + ate = 光，比喻明白 → 阐述

7. e(@)族

e(@)族中，共有en、em两个成员，表示使动的意思。可能是i(@)族的延伸，im=em，in=en。

  • en：

    例词：

    energy = en 使动 + erg 做 + y = 使…工作 → 能量

    engage

  • em：

    例词：

    empower = em 使动 + power 力量 = 使…有力量 → 授权

    empirical

8. ex族

ex族中，虽然成员比较多，但都是表示向外的含义。

  • e：

    例词：

    evade = e 向外 + vad 走 + e = 向外走 → 逃避

    elapse

    evolution

  • ex：

    例词：

    exit = ex 向外 + it 走 = 出口

    export = ex 向外 + 拿 = 出口

    exclude

  • e(*)：

    和e(@)族不一样，这里表示的是向外的意思。

    例词：

    effect = ef 向外 + fect 做 = 做给别人看 → 效果

9. trans

trans的意思是穿过，改变。

例词：

transform = trans 改变 + form 形状 = 变形

translate

transit

10. re

re的意思是往回、再次的意思。

例词：

report = re 往回 + port 拿 = 从现场往回拿东西 → 报道

review = re 再次 + view 看 = 复习

11. d(@)族

d(@)族有很多成员，而且含义也比较复杂，包括否定、分离、向下的含义。

  • de：

    例词：

    deform = de 分离/否定 + form 形状 = 分离形状 = 使…成畸形

    design = de 向下 + sign 写，标记 = 设计

  • di：

    例词：

    direct = di 分开 + rect 正，直 = 直接，指挥

  • dis：

    例词：

    dislike = dis 否定 + like 喜欢 = 不喜欢

    distribute = dis 分离，分开 + tribut 给 + e = 分配

  • dys：

    同dis。

    例词：

    dysfunction = dys 否定 + function 功能 = 功能紊乱

  • di(*)：

    例词：

    difference = dif 分开 + fer 拿 + ence = 分开拿 → 不同

    dissident = dis 分开 + sid 坐 + ent 人或物 = 分开坐的人 → 持不同意见的人

  • is：

    同dis。

    例词：

    island = is 分开 + land 陆地 = 与陆地分开 → 岛屿

  • dia：

    由分开引申为相对，贯穿的意思。

    例词：

    diagram = dia 贯穿 + gram 写 = 将事物贯穿描写清楚 → 图表

    dialog

12. a(@)族

a(@)族成员不少，含义也比较冲突，包含强调语气，可以理解为去(go)的意思，也包含否定的意思。

  • a：

    例词：

    atom = a 否定 + tom 切 = 不能切 = 原子

    available = a 强调语气 + vail 力量 + able = 有力的 → 可获得的

  • ad：

    例词：

    adopt = ad 去 + opt 选择 = 采取，接受

    adverse = ad 否定 + vers 转 = 不好的转动 → 对着干，不利的

  • a(*)：

    例词：

    apply = ap 去 + ply 满 = 吩咐人去完成事情 → 应用，涂

  • ab：

    例词：

    absolute = ab 否定 + solute 溶解 = 不溶解 → 固体 → 绝对的

  • abs：

    例词：

    abstract = abs 强调 + tract 抽取 = 抽出来 → 摘要

13. su族

su族成员也不少，但都是表示向下，在下的意思，或者是自下向上的引申义。

  • su：

    例词：

    suspect = su 自下向上 + spect 看 = 自下向上打量 → 怀疑

  • sub：

    例词：

    subway = sub 在下 + way 路 = 地铁

  • sup：

    例词：

    supply = sup 自下向上 + ply 满 = 供应

  • sus：

    例词：

    sustain = sus 在下 + tain 支持 = 支撑

  • su(*)：

    例词：

    sufficient = suf 在下 + fic 做 + i + ent = 在下方不停工作 → 充分的

14. super族

super的意思是超级、在上的意思，还有个简写sur，也表示相同意思。

  • super：

    例词：

    superstar = super 超级 + star 星星 = 巨星

  • sur：

    例词：

    surface = sur 在上 + face 面 = 上面 → 表面

15. o(@)族

o(@)族共包含ob、o(*)，与a(@)相似，既能表示强调语气，也能表示否定的意思。

  • ob：

    例词：

    obvious = ob 强调 + vi 路 + ous = 在路上很显眼的东西 → 明显

    obstacle = ob 否定 + st 站 + acle = 反着站 → 障碍

  • o(*)：

    例词：

    occur = oc 强调 + cur 跑 = 跑出来 → 出现

    oppose = op 否定，相反 + pos 放 + e = 反着放 → 反对

16. neg族

neg族包含ne与neg，最常见的词就是negative，表示否定的意思。

  • neg：

    例词：

    negative = neg 否定 + ative = 否定的

    neglect

  • ne：

    例词：

    necessary = ne 否定 + cess 坐 + ary = 坐不下来 → 说明有必须做的事情做 → 必须

17. un

un表示否定。

例词：

unhappy = un 否定 + happy 开心 = 不开心

unnecessary

unprofessional

18. 数词族

  • tri：

    表示三。

    例词：

    triangle = tri 三 + angle 角 = 三角形

  • bi：

    表示二。

    例词：

    bicycle = bi 二 + cycle 圆 = 两个圆 → 两个轮子 → 自行车

  • du：

    表示二。

    例词：

    duplicate = du 二 + plic 折 + ate = 对折 → 相等的大小 → 相等 → 重复

19. anti

anti表示反对。

例词：

antiscience = anti 反对 + science 科学 → 反对科学的

20. multi

multi表示多。

例词：

multiply = multi 多 + ply 折 = 多折 → 多倍，乘 （乘法是多次加法得来的）

21. poly

poly表示多，也有相同形式的词根表示贩卖。

例词：

polynomial = poly 多 + nom 项 + i + al = 多项式

22. semi

semi表示半。

例词：

semicircle = semi 半 + circle 圆 = 半圆

23. bene

bene表示好。

例词：

benefit = bene 好 + fit 做 = 受益

24. eu

eu也表示好。

例词：

eulogy = eu 好 + log 话 + y = 好话 → 颂扬

25. mal

mal表示坏。

例词：

malformation = mal 坏 + formation 形状 = 畸形

26. epi

epi表示在上。

例词：

epidemic = epi 在上 + dem 人民 + ic = 在人民之上 → 流行，传染病

27. inter族

inter族有inter、intel、enter三个成员。表示在……之间的意思。

  • inter：

    例词：

    international = inter 在……之间 + nation 国家 + al = 国家间的，国际的

  • intel：

    例词：

    intellect = intel 在……之间 + lect 选 = 选择出来 → 有智慧的 → 智慧

  • enter：

    例词：

    entertainment = enter 在……之间 + tain 拿，持 + ment = 在……之间互相传递 → 觥筹交错 → 娱乐

28. intr(@)族

intr(@)族有intro、intra两个成员。表示向内的意思，与in相似，但这里表示的意思更抽象。

  • intro：

    例词：

    introspect = intro 向内 + spect 看 = 内省 （区分于inspect）

  • intra：

    例词：

    intranet = intra 内部的 + net 网络 = 内部网

29. extr(@)族

extr(@)族有extro、extra两个成员。表示向外的意思，与ex相似，但这里表示的意思更抽象。

  • extro：

    例词：

    extrovent = extro 向外 + vent 来 = 外向

  • extra：

    例词：

    extra 额外的

30. retro

retro表示向内的意思，与re相似，但这里表示的意思更抽象。

  • 例词：

    retrospect = retro 向内 + spect 看 = 回顾 （区别于respect）

31. contr(@)族

contr(@)族有contro、contra两个成员。表示相反的意思。

  • contro：

    例词：

    controversy = contro 反 + vers 转 + y = 反着来 → 争论

  • contra：

    例词：

    contradict = contra 反 + dict 话 = 反驳
