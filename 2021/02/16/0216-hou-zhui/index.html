单词系列（2）——后缀基操篇

单词 词缀
英语
发布日期:   2021-02-16
更新日期:   2021-02-16
文章字数:   2.3k
阅读时长:   9 分
阅读次数:  

hello，今天继续更新第二篇博客。前一篇博客说的是前缀部分。前缀更多决定单词的部分含义，而后缀更多决定单词的词性。

基操篇

1. e

名词或动词后缀。

注：该篇博客中的或，是高中数学中的命题或，不是互斥或。即e表示的词性可以是名词，也可以是动词，也可以即是动词又是名词。

例词：

divide = divid + e

v. （使）分开，分成；分配，分担；分隔；（道路）分叉；使产生分歧；（数）除，除以
n. 分歧，差异；分水岭；界限

2. ed

形容词后缀。表示……的，可能带有被动含义。也可能是动词后缀，表示某动词的过去式、过去分词形式。

注：该篇博客中的或，是高中数学中的命题或，不是互斥或。即e表示的词性可以是名词，也可以是动词，也可以即是动词又是名词。

例词：

divide = divide + ed

adj. 分裂的；分开的；有分歧的
v. 分开（divide的过去分词）；分离

3. (@)ry族

(@)ry族的@共包含4个元音字母a、o、u、e，组合起来就是ary、ory、ury、ery。它们是形容词或名词（比较常见）后缀。

例词：

factory = fact + ory

n. 工厂；制造厂；代理店

diary = di + ary

n. 日志，日记；日记簿

century = cent + ury

n. 世纪，百年；（板球）一百分

mystery = myst + ery

n. 秘密，谜；神秘，神秘的事物；推理小说，推理剧；

4. (@)m族、(@)n族

(@)m族共包含im、um、em；(@)n族共包含an、on。是名词后缀。

例词：

minim = min + im

n. 量滴（液量最小单位）；极小的东西；二分音符
adj. 最小的；微小的
(注：可能会有少量词有延伸的词性，需要单独注意)

calcium = calc + i + um

n. [化学] 钙

可以看到，碱金属、碱土金属等金属的后缀都是um，说明这个后缀用于表示金属元素。

pedestrian = ped + estr + i + an

adj. 徒步的；缺乏想像力的
n. 行人；步行者

an有人的意思。例如christian，european等等。

electron = electr + on

n. 电子

on有小的意思。电子是微粒，所以用了on这个后缀。例如plankton，表示浮游生物。

5. (@)l族

(@)l族共包含al、el、il、ul。是名词或形容词后缀。

例词：

survival = sur + viv + al

n. 幸存，残存；幸存者，残存物

novel = nov + el

adj. 新奇的；异常的
n. 小说

tendril = tendr + il

n. 卷须；蔓；卷须状物

vocabulary = vocab + ul + ary

particular = part + ic + ul + ary

spectular = spect + ul + ar

ul一般不做后缀，而是充当连接成分，连接前一词根和后一词根。

6. (@)ble族、(@)bility族

(@)ble族共包含ible、able、ile。表示能……的，可……的。

(@)bility族共包含ibility、ability、ility。表示能力。

这两个族之间是存在对应关系的。现在拿ible举例子。

ible + ity = iblity， 为了方便拼读，在bl中间增加i，就变成了ibility。

able + ity = ablity， 为了方便拼读，在bl中间增加i，就变成了ability。

ile + ity = ility。

例词：

visible = vis + ible

visibility = vis + ibility

liable = ly + able

liability = ly + ability

mobile = mob + ile

mobility = mob + ility

7. ing

形容词后缀。或是名词后缀，表示动名词。或是动词，表示现在分词。

例词：

amazing = amaze + ing

adj. 令人惊异的
v. 使吃惊（amaze的ing形式）

8. ly

形容词或副词后缀。

例词：

happily = happy + ly

adv. 快乐地；幸福地；幸运地；恰当地

friendly = friend + ly

adj. 友好的；亲切的；支持的；融洽的，和睦的

9. (@)r族

(@)r族共包含er、or、ar、our、ee、ess、ress七个成员。其中，最常见的er表示人或物，or也是表示人或物。ee也是表示人或物，但是有被动含义。ar、our就没有太固定的意思，大致表示名词后缀。ess、ress表示雌性的人或动物。

例词：

employer = em + ploy + er

n. 雇主，老板

employee = em + ploy + ee

n. 雇员；从业员工

professor = pro + fess + or

n. 教授；教师；公开表示信仰的人

professor = pro + fess + or

n. 教授；教师；公开表示信仰的人

amour = am + our

n. 恋情；偷情，奸情

particular = part + ic + ul + ar

adj. 特别的；详细的；独有的；挑剔的
n. 详细说明；个别项目

actress = act + ress
比较一下 actor = act + or

lioness = lion + ess

10. ous

形容词后缀。

例词：

delicous = delic + ous

adj. 美味的；可口的

11. ize/ise

动词后缀，有表示使动的含义。

例词：

realize = real + ize

vt. 实现；认识到；了解；将某物卖得，把(证券等)变成现钱；变卖

12. (@)te族

(@)te族共包含ate、ute、ite三个成员。常用表示动词，也表示形容词。

例词：

create = cre + ate

vt. 创造，创作；造成

dynamite = dynam + ite

n. 炸药；具有潜在危险的人（或物）
vt. 炸毁
adj. 极好的

execute = ex + (s)ec + ute

vt. 实行；执行；处死

13. ion

名词后缀。

例词：

creation = cre + ate + ion

n. 创造，创作；创作物，产物

14. (@)nt族、(@)nc(@)族

(@)nt族共包含ant、ent。形容词后缀或名词后缀。如果是名词后缀，可能表示人或物，类似于er。

(@)nc(@)族共包含ance、ence、ancy、ency。名词后缀。

它们之间存在对应关系。

ant → ance/ancy

ent → ence/ency

例词：

attendant = attend + ant

attendance = attend + ance

different = diff + er + ent

difference = diff + er + ence

hesitant = hesit + ant

hesitancy = hesit + ancy

hesitance = hesit + ance

permanent = per + man + ent

permanence = per + man + ence

15. a

名词后缀，可能表示一类物品（集合）。

例词：

formula = form + ul + a

n. [数] 公式，准则；配方；婴儿食品

16. s

名词后缀，表示复数。

例词：

books = book + s

17. ia

名词后缀，表示疾病。

例词：

nostalgia = nost(=nest) + alg + ia

n. 乡愁；怀旧之情；怀乡病
这个解释真的很贴切了(doge)

18. it、id

形容词后缀。

例词：

explicit = ex + plic + it

valid = val + id

19. (@)ty族

(@)ty族共包含ity、ety、ty三个成员。名词后缀。

例词：

activity = act + ive + ity

n. 活动；行动；活跃

variety = var + i + ety

n. 多样；种类；杂耍；变化，多样化

certainty = certain + ty

n. 必然；确实；确实的事情

20. ist

名词后缀，表示学者。

例词：

biologist = bi + o + log + ist

n. 生物学家

21. ic、ique

形容词后缀。

例词：

biologic = bi + o + log + ic

adj. 生物的；生物学的

有一个与它读音相近的后缀ique，它是名词后缀。

例词：

technique = techn + ique

n. 技巧，技术；手法

22. logy

名词后缀，表示学说，由log + y组合而成。

例词：

biology = bi + o + logy

n. （一个地区全部的）生物；生物学

23. ism

名词后缀，表示……主义。

例词：

biologism = bi + o + log + ism

n. 生物主义

也有asm这个后缀，用于表示一些抽象的概念。

orgasm = org + asm

n. 极度兴奋 （还有别的意思，这里删减了）

24. ics

名词后缀，表示学说，与logy类似。

例词：

statics = stat + ics

n. [力] 静力学

25. ure

名词后缀，偶尔表示动词。

例词：

structure = struct + ure

n. 结构；构造；建筑物
vt. 组织；构成；建造

26. ment

名词后缀。

例词：

apartment = a + part + ment

n. 公寓；房间

27. (@)de族

(@)de族共包含ade、ude。名词后缀。

例词：

parade = par + ade

n. 游行；阅兵；炫耀；行进；阅兵场
vt. 游行；炫耀；列队行进
vi. 游行；炫耀；列队行进

altitude = alt + it + ude

n. 高地；高度；[数] 顶垂线；（等级和地位等的）高级；海拔

28. fy

动词后缀。

例词：

purify = pur + i + fy

vt. 净化；使纯净
vi. 净化；变纯净

29. age

名词后缀。

例词：

passage = pass + age

n. 一段（文章）；走廊；通路；通过；旅程，行程；船费

30. (@)cle族

(@)cle族共包含icle、acle。名词后缀。

例词：

obstacle = ob + st + acle

n. 障碍，干扰，妨碍；障碍物

article = art + icle

n. 文章；物品；条款；[语] 冠词
vt. 订约将…收为学徒或见习生；使…受协议条款的约束
vi. 签订协议；进行控告

31. ice

名词后缀。

例词：

novice = nov + ice

n. 初学者，新手

32. acy

名词后缀。

例词：

diplomacy = di + plom + acy

n. 外交；外交手腕；交际手段

33. ish

动词或形容词后缀。

例词：

publish = publ + ish

vt. 出版；发表；公布
vi. 出版；发行；刊印

selfish = self + ish

adj. 自私的；利己主义的

34. (@)sis族

(@)sis族共包含asis、osis。名词后缀，与疾病相关。

例词：

hemostasis = hemo(血) + st(站，停止) + asis

n. [医] 止血；止血法

neurosis = neur + osis

n. [心理] 神经症；神经衰弱症

35. (@)et族

(@)et族共包含et、let、ette。名词后缀，用来表示较小的物品。与on不同的是，on一般表示的是微观的粒子或者是生物，et更多表示生活中的物品。

例词：

cigarette = cigar(雪茄) + ette

n. 香烟；纸烟

coverlet = cover + let

n. 被单；床罩

circlet = circl + et

n. 小圈；戒指，饰环

36. y

名词后缀。

例词：

beauty = beaut + y

n. 美；美丽；美人；美好的东西

37. ine

名词或形容词后缀。

例词：

marine = mar + ine

adj. 船舶的；海生的；海产的；航海的，海运的
n. 海运业；舰队；水兵；（海军）士兵或军官

38. less

形容词后缀，表示没有。

例词：

homeless = home + less

adj. 无家可归的

39. ness

名词后缀。

例词：

necessariness = ne + cess + ary + ness

n. 必要性

40. ful

形容词后缀，表示多(full)。

例词：

beautiful = beaut + i + ful

adj. 美丽的；出色的；迷人的

41. ship

名词后缀，表示抽象含义。

例词：

relationship = re + lat + ion + ship

n. 关系；关联

42. ese

名词后缀，表示某国家的人或语言。

43. hood

名词后缀，表示抽象含义，地点或一段时间等等。

例词：

childhood = child + hood

n. 童年时期；幼年时代
